# LangChain Gemini Search Agent

## Overview
This project demonstrates a **search‑assistant** built with the **LangChain** framework and **Google Gemini** (gemini‑1.5‑flash) model.  It:

- Loads PDF, TXT and markdown documents from a local `data/` folder.
- Creates a **FAISS** vector index using Gemini embeddings.
- Wraps the index in a `ConversationalRetrievalChain` to answer natural‑language queries.
- Provides a sleek **Streamlit** UI with a black‑and‑white (dark) theme for interactive chat.
- Includes a Dockerfile (optional) for containerised deployment.

## Quick start (local)
```powershell
# 1. Clone the repo (already done) and cd into the project folder
cd C:\Users\devan\.gemini\antigravity\scratch\langchain_search_agent

# 2. (Optional) create a virtual environment
python -m venv .venv
.venv\Scripts\activate

# 3. Install dependencies
pip install -r requirements.txt

# 4. Add your Gemini API key to .env (already created)
#    GEMINI_API_KEY=YOUR_KEY

# 5. Place documents in the `data/` directory.

# 6. Run the UI
streamlit run src/main.py
```
Open the printed **Local URL** (e.g. `http://localhost:8501`) in a browser.

## Docker (optional)
```dockerfile
FROM python:3.12-slim
WORKDIR /app
COPY . /app
RUN pip install --no-cache-dir -r requirements.txt
EXPOSE 8501
CMD ["streamlit", "run", "src/main.py", "--server.port", "8501"]
```
Build and run:
```bash
docker build -t langchain-gemini .
docker run -p 8501:8501 langchain-gemini
```
